# Toastingway

![Usage](./screenshot.png)

A simple plugin to show some native toasts for items, crystals, or gil.

## Main Points

* Shows toasts for 
  * Items
  * Crystals
  * Gil
  * MGP
  * GC Seals
  * Wolf Marks
  * Allied Seals
* Toasts can be toggled on and off for broad categories.
